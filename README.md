# Sistema de Controle de Automóveis

API para controle de utilização de automóveis de uma empresa.

## Funcionalidades

- Cadastro de automóveis
- Cadastro de motoristas
- Controle de utilização de automóveis

## Tecnologias

- Node.js
- Express.js
- Jest (testes)
- ESLint + Prettier (padronização)

## Instalação

1. Clone o repositório:
```bash
git clone [url-do-repositorio]
cd auto_controll
```
